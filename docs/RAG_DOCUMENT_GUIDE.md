# D&D 5E 文档准备指南

## 📚 需要的文档

为了构建完整的 RAG 知识库，请准备以下文档：

### 1. 核心规则书

```
docs/dnd5e/
├── player_handbook.md          # 玩家手册
├── monster_manual.md            # 怪物图鉴
└── dungeon_master_guide.md      # DM 指南
```

### 2. 文档格式要求

**推荐格式：Markdown**

原因：
- ✅ 易于按章节切分
- ✅ 保留结构信息（标题、列表）
- ✅ 向量检索效果好

**示例格式（怪物图鉴）**：

```markdown
## 地精 (Goblin)

**类型**: 人形生物  
**挑战等级**: 1/4 (50 XP)

### 属性
- **护甲等级**: 15（皮甲、盾牌）
- **生命值**: 7 (2d6)
- **速度**: 30尺

**力量** 8 (-1)  **敏捷** 14 (+2)  **体质** 10 (+0)  
**智力** 10 (+0)  **感知** 8 (-1)   **魅力** 8 (-1)

### 动作

**弯刀**. _近战武器攻击_: +4 命中，触及5尺，一个目标。  
_伤害_: 5 (1d6 + 2) 挥砍伤害。

**短弓**. _远程武器攻击_: +4 命中，射程80/320尺，一个目标。  
_伤害_: 5 (1d6 + 2) 穿刺伤害。

### 描述

地精是狡猾的绿皮小人，身高约3到3.5英尺。它们喜欢在黑暗中伏击，经常成群出现...

---

## 骷髅 (Skeleton)

**类型**: 不死生物  
**挑战等级**: 1/4 (50 XP)

...
```

### 3. 如果有 PDF 文档

如果你有 PDF 格式的官方书籍：

**转换方法**：

```bash
# 1. 安装 pdftotext (推荐)
# Windows: 下载 Xpdf tools

# 2. 转换为文本
pdftotext "Player's Handbook.pdf" player_handbook.txt

# 3. 手动整理为 Markdown（或用脚本）
```

**注意**：
- PDF 转换可能丢失格式
- 建议人工校对关键怪物/规则

### 4. 简化方案（快速开始）

如果没有完整文档，可以先创建简化版：

```markdown
攻击: 巨斧 +5 (1d12+3)

```

## 📁 文件存放位置

```bash
# 在项目根目录创建
mkdir -p docs/dnd5e

# 放入文档
docs/dnd5e/
├── monster_manual.md        # 必需
├── player_handbook.md       # 可选（用于技能、法术检索）
└── custom_content.md        # 自定义内容
```

## 🚀 导入到向量数据库

准备好文档后，运行：

```bash
# 1. 启动服务器
go run ./cmd/server/main.go

# 2. 索引文档（待实现的 API）
curl -X POST http://localhost:8080/api/v1/rag/index \
  -H "Content-Type: application/json" \
  -d '{
    "file": "docs/dnd5e/monster_manual.md",
    "source": "monster_manual"
  }'
```
